# Dossiers

Create one directory per story.

Suggested structure:

```text
dossiers/
  YYYY-MM-DD/
    <slug>/
      DOSSIER.md
      sources.json
      claims.csv
      notes.md
      draft.md
      verify.md
      digest.md
      publish.md
```

Rules:
- every publishable story must have a dossier
- do not publish from chat history alone
- the dossier is the durable editorial record
