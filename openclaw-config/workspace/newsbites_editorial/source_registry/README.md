# Source Registry

Use this directory for durable beat-level source lists.

Suggested structure:
- `ai.md`
- `finance.md`
- `global-politics.md`
- `trends.md`

Each registry file should include:
- official institutions
- regulators
- company IR/newsrooms
- trusted beat reporters
- trade publications
- known weak or hype-prone sources to treat cautiously

Rule:
- registry files are for source quality and coverage memory
- they are not article dossiers
